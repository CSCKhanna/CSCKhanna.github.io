# Pillar 1: Teaching and Learning

Interactive University of Rhode Island themed HTML presentation for Pillar 1: Teaching and Learning.

## Local Preview

Open `index.html` in a web browser.

## Navigation

- Left/right arrows move through the main presentation.
- Up/down arrows move through drill-down slides.
- `Esc` opens the overview.
- `F` toggles fullscreen.
- `?` opens shortcut help.
